# Contractor Directory MVP

A local contractor/home services directory website.

## Features
- Search by radius around user's location
- Search by services (plumber, electrician, etc.)
- Search by company names
- Basic listing display

## Tech Stack
- Vanilla HTML/CSS/JavaScript
- Geolocation API for user location
- Deployable to Vercel

## Deployment

### Option 1: Vercel (Recommended)
1. Push this repo to GitHub
2. Go to https://vercel.com
3. Import the repository
4. Vercel will auto-detect and deploy

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel
```

## Development
Simply open `index.html` in a browser, or serve locally:
```bash
npx serve .
```
